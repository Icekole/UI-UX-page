## 📝 如何修改内容

### 修改个人信息
编辑 `data/portfolio.json`：
```json
{
  "name": "你的名字 | UI/UX 设计师",
  "headerTaglineOne": "你好 👋",
  "headerTaglineTwo": "我是 XXX",
  // ...
}
```

### 修改作品集
编辑 `data/portfolio.json` 中的 `projects` 数组：
```json
"projects": [
  {
    "id": "1",
    "title": "项目名称",
    "description": "项目描述",
    "imageSrc": "图片 URL",
    "url": "项目链接"
  }
]
```

### 添加更多资源
编辑 `data/portfolio.json` 中的 `resources` 数组：
```json
"resources": [
  {
    "category": "分类名称",
    "links": [
      {"name": "网站名", "url": "https://...", "desc": "描述"}
    ]
  }
]
```

## 📸 替换项目图片

当前使用的是 Unsplash 占位图片，建议替换成你的设计作品：

1. 将图片放入 `public/images/` 文件夹
2. 修改 `data/portfolio.json` 中的 `imageSrc`：
```json
"imageSrc": "/images/your-project.jpg"
```
