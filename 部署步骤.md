# 🚀 部署到 GitHub 完整步骤

## 第一步：在 GitHub 创建仓库

1. **访问** https://github.com/new
2. **填写信息**：
   - Repository name: `UI-UX-Site`
   - Description: "UI/UX 设计师个人作品集网站"
   - Public（公开）或 Private（私有）都可以
   - ❌ **不要** 勾选 "Initialize this repository with a README"
3. **点击** "Create repository"

---

## 第二步：推送代码到 GitHub

打开命令行（PowerShell 或 CMD），执行以下命令：

```bash
# 进入项目目录
cd E:\Soft\Ai\portfolio

# 初始化 git（如果已初始化可跳过）
git init

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit"

# 重命名分支为 main
git branch -M main

# 关联远程仓库（替换 YOUR_GITHUB_USERNAME 为你的 GitHub 用户名）
git remote add origin https://github.com/YOUR_GITHUB_USERNAME/UI-UX-Site.git

# 推送到 GitHub
git push -u origin main
```

**替换示例**：
如果你的 GitHub 用户名是 `zhangsan`，则命令为：
```bash
git remote add origin https://github.com/zhangsan/UI-UX-Site.git
git push -u origin main
```

---

## 第三步：配置 GitHub Pages

推送成功后：

1. **访问你的仓库** https://github.com/YOUR_GITHUB_USERNAME/UI-UX-Site
2. **点击** "Settings"（设置）标签
3. **左侧菜单找到** "Pages"
4. **在 Build and deployment 下**：
   - Source: 选择 **GitHub Actions**
5. **等待自动部署**（大约 2-5 分钟）

---

## 第四步：查看部署进度

1. **点击** "Actions" 标签
2. **查看** 最新的 workflow 运行状态
3. **等待** 所有步骤变成绿色 ✓

---

## 第五步：访问你的网站

部署成功后，访问你的网站：

```
https://YOUR_GITHUB_USERNAME.github.io/UI-UX-Site/
```

---

## 🎯 快速部署脚本

创建一个 `deploy.bat` 文件（已创建），双击运行，输入你的 GitHub 用户名即可。

或者手动执行：

```batch
@echo off
set /p USERNAME="请输入你的 GitHub 用户名："
git remote add origin https://github.com/%USERNAME%/UI-UX-Site.git
git push -u origin main
pause
```

## ❓ 常见问题

### 1. 提示 "Repository not found"
**解决**：确保已在 GitHub 创建仓库

### 2. 提示 "Permission denied"
**解决**：检查远程仓库地址是否正确

### 3. 推送成功但网站无法访问
**解决**：
- 等待 5-10 分钟
- 检查 Actions 中部署是否成功
- 确认 Pages 设置中 Source 选择了 GitHub Actions

### 4. 图片不显示
**解决**：
- 确保图片放在 `public/images/` 目录
- 使用绝对路径：`/images/xxx.jpg`

## 🎨 定制内容清单

在部署前，建议先修改以下内容：

1. **个人信息** - `data/portfolio.json`
   - name（你的名字）
   - headerTaglineTwo（个人简介）
   - socials（社交链接）
   - aboutpara（关于我）

2. **作品集** - `data/portfolio.json`
   - projects（项目标题、描述、图片）

3. **资源导航** - `data/portfolio.json`
   - resources（添加你喜欢的网站）
