# 个人门户网站交互文档

## 一、页面入口

- 用户通过公网地址或本地 HTML 文件进入个人门户，默认展示首页。
- 顶部导航提供四个主视图入口：首页、作品、文章、工具箱。
- 顶部“联系我”按钮可在任意视图打开联系抽屉。
- 首页 CTA 提供三个入口：查看作品、阅读文章、联系我。

## 二、页面规则

### 1. 首页

- 首页首屏展示“产品设计与 AI 原型实践”作为主标题。
- 副标题展示“从 UI/UX 到产品方案落地”。
- 信息标签展示产品设计、产品管理、UI/UX 转产品、AI 工作流、武汉 / 远程协作。
- 首页文案说明能力方向：从用户体验、业务目标到落地协作，转化为产品方案、交互原型和可执行文档。
- 能力摘要展示 UI/UX、PM、AI 三个方向。

### 2. 作品

- 作品视图默认展示全部作品。
- 筛选项包括：全部、产品项目、设计作品、方法论。
- 筛选项互斥激活，同一时间只允许一个筛选项高亮。
- 作品卡片展示作品封面图、类型、标题、摘要、标签和结果说明。
- 作品内容基于 2023 UI/UX 作品集整理，覆盖高校一站式服务大厅、业务流程与功能架构、体验改版问题分析、武汉大学樱花预约、招生概况数据大屏和作品集结构化整理。
- 作品详情通过二级页面展示大图、图片说明、关键指标、项目背景、我的工作、关键设计和结果沉淀。
- 作品详情页顶部提供“返回作品”按钮，点击后回到作品列表，并保持当前作品筛选状态。
- 作品内容必须按公开展示口径脱敏，不展示不可公开流程、真实业务敏感数据和未授权材料。

### 3. 文章

- 文章视图展示周一至周五 5 张固定晨读封面海报。
- 晨读封面卡片采用左图右文结构：左侧为用户提供的工作日封面海报，右侧顶部为今日晨读描述，下方为当天日期。
- 晨读封面图片来自 `assets/reading/` 本地资产目录。
- 文章标题、列表摘要、日期、趋势标题、趋势摘要、主题标签、阅读时长和详细内容作为文章数据保留，用于详情页展示。
- 当前文章详情通过二级页面展示。
- 文章详情页顶部提供“返回文章”按钮，点击后回到文章列表。
- 晨读封面卡片支持鼠标点击和键盘 Enter、Space 打开详情。

### 4. 工具箱

- 工具箱默认展示设计分类。
- 分类包括：设计、产品、AI、研发。
- 分类互斥激活，同一时间只允许一个分类高亮。
- 资源卡片展示工具名称、用途说明和打开提示。
- 点击资源卡片后在新窗口打开外部链接，当前门户页面保持原状态。

## 三、按钮跳转

| 入口 | 点击结果 |
| --- | --- |
| 顶部“首页” | 切换到首页视图，首页导航高亮 |
| 顶部“作品” | 切换到作品视图，作品导航高亮 |
| 顶部“文章” | 切换到文章视图，文章导航高亮 |
| 顶部“工具箱” | 切换到工具箱视图，工具箱导航高亮 |
| 顶部“联系我” | 打开联系抽屉 |
| 首页“查看作品” | 切换到作品视图 |
| 首页“阅读文章” | 切换到文章视图 |
| 首页“联系我” | 打开联系抽屉 |
| 作品筛选项 | 切换作品分类并刷新作品列表 |
| 作品卡片 | 进入作品详情二级页面 |
| 作品详情“返回作品” | 返回作品列表，保留当前筛选状态 |
| 晨读封面卡片 | 进入文章详情二级页面 |
| 文章详情“返回文章” | 返回文章列表 |
| 工具箱分类 | 切换工具箱资源分类并刷新资源列表 |
| 资源卡片 | 新窗口打开外部链接 |

## 四、详情页与弹窗交互

### 1. 作品详情二级页面

- 点击作品卡片后，页面切换到作品详情二级页面。
- 作品详情页展示作品标题、脱敏详情、作品大图、标签、关键指标和结构化说明。
- 顶部导航保持“作品”高亮。
- 点击“返回作品”后返回作品列表。
- 返回作品列表后，作品筛选项和列表内容保持不变。

### 2. 文章详情二级页面

- 点击晨读封面卡片后，页面切换到文章详情二级页面。
- 文章详情页展示列表摘要标题、日期、趋势标题、趋势摘要、主题标签、阅读时长和详细内容。
- 文章详情内容保持原弹窗中的阅读层次，描述区展示趋势标题和趋势摘要，详细内容区承载 Markdown 正文或结构化分析卡片。
- 顶部导航保持“文章”高亮。
- 点击“返回文章”后返回文章列表。
- 晨读封面卡片支持键盘 Enter、Space 进入详情页。

### 3. 联系抽屉打开

- 点击联系入口后，抽屉展示联系说明、联系邮箱和外部平台入口。
- 抽屉打开后，焦点自动移动到关闭按钮。

### 4. 联系抽屉关闭

- 用户点击关闭按钮后关闭抽屉。
- 用户点击遮罩区域后关闭抽屉。
- 用户按 Escape 键后关闭抽屉。
- 抽屉关闭后，焦点回到打开抽屉前的元素。

## 五、状态规则

- 同一时间只允许一个主视图激活。
- 同一时间只允许一个抽屉打开。
- 顶部导航高亮必须与当前主视图保持一致；处于作品详情页时，顶部导航仍高亮“作品”，处于文章详情页时，顶部导航仍高亮“文章”。
- 作品筛选状态在进入和退出作品详情页后保持不变。
- 晨读封面列表状态在进入和退出文章详情页后保持不变。
- 工具箱分类状态在打开外链后保持不变。
- 移动端下作品、文章、工具箱列表和详情页切换为单列展示。
- 用户开启减少动效时，页面动画和过渡时间降级到最短。

## 六、空状态和异常规则

- 作品为空时展示“作品正在整理中”。
- 当前作品分类无内容时展示“该分类暂无内容”。
- 文章为空时展示“内容即将上线”。
- 工具箱分类无资源时展示“该分类暂无资源”。
- 外链为空时不展示可点击态，并提示“链接暂未配置”。
- 受控资料不允许直接公开下载，必须通过后续沟通、访问码页面、私密链接或 PDF 附件按需提供。
