# 个人门户网站交互文档

## 一、页面入口

- 用户通过公网地址或本地 HTML 文件进入个人门户，默认展示首页。
- 顶部导航提供四个主视图入口：首页、作品、文章、工具箱。
- 顶部“联系我”按钮可在任意视图打开联系抽屉。
- 首页 CTA 提供三个入口：查看作品、阅读文章、联系我。

## 二、页面规则

### 1. 首页

- 首页首屏展示“产品设计与 AI 原型实践”作为主标题。
- 副标题展示“从 UI/UX 到产品方案落地”。
- 信息标签展示产品设计、产品管理、UI/UX 转产品、AI 工作流、武汉 / 远程协作。
- 首页文案说明能力方向：从用户体验、业务目标到落地协作，转化为产品方案、交互原型和可执行文档。
- 能力摘要展示 UI/UX、PM、AI 三个方向。

### 2. 作品

- 作品视图默认展示全部作品。
- 筛选项包括：全部、产品项目、设计作品、方法论。
- 筛选项互斥激活，同一时间只允许一个筛选项高亮。
- 作品卡片展示作品封面图、类型、标题、摘要、标签和结果说明。
- 作品内容基于 2023 UI/UX 作品集整理，覆盖高校一站式服务大厅、业务流程与功能架构、体验改版问题分析、武汉大学樱花预约、招生概况数据大屏和作品集结构化整理。
- 作品详情通过二级页面展示大图、图片说明、关键指标、项目背景、我的工作、关键设计和结果沉淀。
- 作品详情页顶部提供“返回作品”按钮，点击后回到作品列表，并保持当前作品筛选状态。
- 作品内容必须按公开展示口径脱敏，不展示不可公开流程、真实业务敏感数据和未授权材料。

### 3. 文章

- 文章视图展示 Notion CMS 同步的每日晨读、文章分析和历史归档。
- 晨读区域展示当天最新晨读和历史归档，内容来源于 `content/readings/latest.json` 与 `content/readings/index.json`。
- 最新晨读展示日期、趋势标题、趋势摘要和标签；点击“阅读详情”后通过抽屉查看完整晨读。
- 晨读历史按日期倒序展示，点击历史项后读取对应归档 JSON 并打开详情抽屉。
- 文章模块不再使用静态写死文章列表，所有公开内容均从 CMS 同步 JSON 中读取。
- 内容详情通过抽屉展示，不跳转独立文章详情页。

### 4. 工具箱

- 工具箱默认展示设计分类。
- 分类包括：设计、产品、AI、研发。
- 分类互斥激活，同一时间只允许一个分类高亮。
- 工具箱资源来源于 Notion 工具导航 CMS 同步生成的 `content/tools/index.json`。
- 资源卡片展示工具名称、用途说明和打开提示。
- 点击资源卡片后在新窗口打开外部链接，当前门户页面保持原状态。

## 三、按钮跳转

| 入口 | 点击结果 |
| --- | --- |
| 顶部“首页” | 切换到首页视图，首页导航高亮 |
| 顶部“作品” | 切换到作品视图，作品导航高亮 |
| 顶部“文章” | 切换到文章视图，文章导航高亮 |
| 顶部“工具箱” | 切换到工具箱视图，工具箱导航高亮 |
| 顶部“联系我” | 打开联系抽屉 |
| 首页“查看作品” | 切换到作品视图 |
| 首页“阅读文章” | 切换到文章视图 |
| 首页“联系我” | 打开联系抽屉 |
| 作品筛选项 | 切换作品分类并刷新作品列表 |
| 作品卡片 | 进入作品详情二级页面 |
| 作品详情“返回作品” | 返回作品列表，保留当前筛选状态 |
| 晨读“阅读详情” | 打开最新晨读详情抽屉 |
| 晨读历史项 | 读取对应归档内容并打开晨读详情抽屉 |
| 工具箱分类 | 切换工具箱资源分类并刷新资源列表 |
| 资源卡片 | 新窗口打开外部链接 |

## 四、详情页与弹窗交互

### 1. 作品详情二级页面

- 点击作品卡片后，页面切换到作品详情二级页面。
- 作品详情页展示作品标题、脱敏详情、作品大图、标签、关键指标和结构化说明。
- 顶部导航保持“作品”高亮。
- 点击“返回作品”后返回作品列表。
- 返回作品列表后，作品筛选项和列表内容保持不变。

### 2. 详情抽屉打开

- 点击晨读详情入口后，抽屉展示晨读标题、趋势摘要、标签、日期、来源数量和完整正文。
- 点击 CMS 内容入口后，抽屉展示标题、摘要、标签、日期、来源数量和完整正文。
- 点击联系入口后，抽屉展示联系说明、联系邮箱和外部平台入口。
- 抽屉打开后，焦点自动移动到关闭按钮。

### 3. 详情抽屉关闭

- 用户点击关闭按钮后关闭抽屉。
- 用户点击遮罩区域后关闭抽屉。
- 用户按 Escape 键后关闭抽屉。
- 抽屉关闭后，焦点回到打开抽屉前的元素。

## 五、状态规则

- 同一时间只允许一个主视图激活。
- 同一时间只允许一个抽屉打开。
- 顶部导航高亮必须与当前主视图保持一致；处于作品详情页时，顶部导航仍高亮“作品”。
- 作品筛选状态在进入和退出作品详情页后保持不变。
- 工具箱分类状态在打开外链后保持不变。
- 移动端下作品、文章、工具箱列表切换为单列展示，城市竖排标签隐藏。
- 用户开启减少动效时，页面动画和过渡时间降级到最短。

## 六、空状态和异常规则

- 作品为空时展示“作品正在整理中”。
- 当前作品分类无内容时展示“该分类暂无内容”。
- 文章为空时展示“内容即将上线”。
- 晨读 JSON 读取失败时展示“晨读内容读取失败，请稍后刷新”。
- 晨读无已发布内容时展示“暂无已发布晨读”。
- 工具箱 JSON 读取失败或当前分类无资源时展示“暂无已发布工具资源”。
- 外链为空时不展示可点击态，并提示“链接暂未配置”。
- 受控资料不允许直接公开下载，必须通过后续沟通、访问码页面、私密链接或 PDF 附件按需提供。
